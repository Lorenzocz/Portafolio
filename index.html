Sistema de Asitencia para Actividades

Sistema de eventos, es una aplicación utilizada con el objetivo de agilizar la entrada a actividades de eventos en específicos.
​

Esta aplicación se realizó como proyecto de Universidad, misma necesaria para concluir los estudios, esta aplicación la desarrolle para el CFIA, Colegio Federado de Ingenieros y de Arquitectos de Costa Rica, con una duración de práctica de 6 meses.

Dicha aplicación cuenta con lector QR utilizado para validar la asistencia de los participantes, este con la funcionalidad de indicar si la persona puede o no ingresar a una actividad de un evento especifico, ya que existen muchos eventos con muchas actividades disponibles un mismo día.

Covid-19 CR

Covid-19 CR

Es una aplicación personal, que se encarga de llevar un control de todos los casos de covid en Costa Rica.

Esta Aplicación fue desarrollado en Flutter utilizando protón bloc, para manejo de estados y una buena arquitectura, permitiendo esta sea escalable.

No se encuentra en ninguna tienda, ya que Google Play no permite ninguna aplicación referentes al covid-19.

Lotto App

Lotto App es una aplicación móvil encargada de notificar y mostrar los resultados de todos los juegos de lotería en Costa Rica.

Es totalmente gratis y los usuarios reciben información sobre sus juegos de lotería favoritos, mediante la integración de notificaciones push.

Cuenta con una aplicación admin, utilizada para administrar los resultados,Lotto App es un proyecto personal.

Actualmente se encuentra disponible en la tienda de la Play Store.

Tarea-UCR

UCR-Tarea

Es una aplicación personal, encargada de cargar noticias relevantes sobre la universidad, esta tarea se realizo con el objetivo de aprender cada día.

Esta es mi primer aplicativo móvil, el cual desarrolle para aprender a utilizar el framework de flutter y el manejador de estados de bloc, por lo cual carece de una buen arquitectura y malas prácticas de desarrollo, gracias a esta aplicación aprendí a crear aplicativos con buenos patrones de diseños, manejadores de estados y algo muchos más importantes, que sean escalables.
